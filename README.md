# Conversor Binário para Decimal em C

Projeto desenvolvido em C para converter números binários em decimal.

## Sobre o projeto

O programa recebe um número binário digitado pelo usuário, valida se a entrada contém apenas 0 e 1 e, em seguida, realiza a conversão para decimal.

## Funcionalidades

- Leitura de número binário como string
- Validação da entrada
- Conversão de binário para decimal
- Exibição de mensagem de erro para entrada inválida

## Conceitos praticados

- Strings em C
- Laços de repetição (`for`)
- Estruturas condicionais (`if/else`)
- Funções
- Validação de entrada
- Manipulação de caracteres
- Conversão de bases numéricas
